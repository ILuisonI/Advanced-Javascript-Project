Cards Display
This is the card display, you can see the picture's title, the name of the photographer and it's price. If you click on any picture you can see more information about it. If you are an admin you can edit the pictures by clicking the yellow edit button or you can delete them. If you loged in you can add the pictures to your cart.



List Display
This is the list display, you can see the picture's URL, the title and the name of the photographer. If you click on any picture you can see more information about it. If you are an admin you can edit the pictures by clicking on the edit button or you can delete them. If you loged in you can add the pictures to your cart.



Carousel Display
This is the carousel display, you can see the pictures URL and the name of the photographer.
The carousel changes the picture every 3 seconds.
If you click on the right arrow you will go to the next picture and if you click on the left arrow you will go back to the previous picture.
